The Description of a Software Developer

As a software developer, you Research, design, and develop computer and network software/specialized utility programs. Analyze user needs. Develop software solutions. Apply Principles and techniques of computer science, engineering and mathematical analysis. Update software / enhance existing software capabilities. A software developer may work with computer hardware engineers to integrate hardware + software systems, And develop specifications and performance requirements. One may maintain databases within a application area, working independently or coordinating database development as a team member

Tasks as a Software Developer

  1. Analyze information in order to determine, recommend, and/or plan installation of a new system, or even a modification to the existing system
  2. Analyze the needs of the user and software requirements to determine feasibility of design with in the time and cost constraints
  3. Confer with data processing or project managers to obtain information for limitations or capabilities for data processing projects
  4. Consult with customers or other departments on project status, proposals, or technical issues, such as software system or design maintenance
  5. Coordinate installation of software system.
  6. Train users to use new/modified equipment
